# AI-Powered Real Estate Feasibility Analysis System

## Overview

This project is an AI-powered real estate feasibility and zoning analysis system developed using n8n.

The workflow automatically:

- Retrieves zoning and parcel information
- Analyzes nearby amenities and urban facilities
- Collects real estate market data
- Generates AI-powered feasibility reports
- Estimates project value and ROI
- Stores analysis results in Google Sheets

## Technologies

- n8n
- Google Gemini
- Google Places API
- SerpAPI
- Google Sheets API
- JavaScript

## Use Cases

- Land investment analysis
- Real estate feasibility studies
- Urban planning support
- Investment decision support systems
